# BOSS 直聘自动刷简历工具

这是一个面向 Windows 的 BOSS 直聘候选人简历自动浏览脚本。程序通过模拟鼠标、键盘和剪贴板操作，依次浏览“推荐牛人”页面中的候选人；当简历正文命中指定关键词时，可自动通过邮件转发简历。

> 本项目依赖固定屏幕坐标操作网页。网页布局、浏览器缩放、窗口位置或屏幕分辨率变化，都可能导致误点。请先在测试环境中校准坐标，并遵守 BOSS 直聘的服务条款、招聘合规要求及个人信息保护规定。

## 主要功能

- 启动时输入一个或多个简历关键词，以及转发失败时使用的备用邮箱
- 自动点击鼠标所在位置的第一位候选人
- 复制候选人详情页文本并进行不区分大小写的关键词匹配
- 命中任意关键词时，按预设坐标执行邮件转发
- 每位候选人随机停留 12–18 秒，并随机上下滚动
- 使用键盘右方向键切换至下一位候选人
- 每浏览 100 位候选人后按 `F5` 刷新页面
- 支持 `Esc` 停止和空格键暂停/继续
- 将运行信息写入 `logs/simple_brush.log`

## 运行环境

- Windows 10/11
- Python 3.10 或更高版本
- Microsoft Edge
- 建议使用 1920 × 1080 分辨率；当前转发坐标按该分辨率校准

## 安装

首次使用时双击：

```text
setup.bat
```

脚本会创建 `venv` 虚拟环境并安装 `requirements.txt` 中的依赖。

也可以在命令行中手动安装：

```powershell
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
```

## 使用方法

1. 使用 Edge 登录 BOSS 直聘，并打开“推荐牛人”页面。
2. 双击 `start.bat`。
3. 输入触发转发的关键词；多个关键词使用英文分号 `;` 分隔，留空则禁用自动转发。
4. 启用转发时，输入备用邮箱。
5. 将鼠标移动到第一位候选人的卡片上，并保持页面窗口位置不变。
6. 倒计时结束后，程序开始自动浏览。

运行期间：

- 按 `空格键`：暂停或继续
- 按 `Esc`：安全停止

## 命令行参数

除了交互输入，也可以直接传入参数：

```powershell
venv\Scripts\python.exe simple_brush.py --keywords "Python;数据分析" --email "backup@example.com" --auto
```

| 参数 | 说明 |
| --- | --- |
| `--keywords` | 触发转发的关键词，多个关键词用英文分号分隔 |
| `--email` | “最近联系”未填入邮箱时使用的备用邮箱 |
| `--auto` | 跳过交互输入，直接使用命令行参数 |
| `--no-forward` | 当前版本会解析该参数，但尚未将其接入转发开关；需要禁用转发时请将关键词留空 |

## 工作流程

```text
输入关键词和备用邮箱
        ↓
手动将鼠标放到第一位候选人卡片
        ↓
程序点击并打开候选人详情
        ↓
复制详情页文本并检测关键词
        ↓
命中关键词 ── 是 ──→ 执行邮件转发
        │
        否
        ↓
随机滚动并停留 12–18 秒
        ↓
按右方向键浏览下一位
        ↓
每 100 位按 F5 刷新并重新开始
```

## 坐标校准

邮件转发依赖 `simple_brush.py` 顶部“转发功能配置”中的固定坐标，包括：

- 转发图标
- “邮件转发”标签页
- 邮箱输入框
- 最近联系邮箱
- 转发按钮
- 候选人详情页中心位置

如果界面发生变化，请修改对应的 `*_X` 和 `*_Y` 常量。校准时应固定以下条件：

- Edge 窗口大小和位置
- Windows 显示缩放比例
- 浏览器页面缩放比例
- BOSS 直聘页面布局

建议先使用不重要的测试数据逐个验证点击位置；如出现误点，立即按 `Esc` 停止。

## 日志与隐私

- 运行日志：`logs/simple_brush.log`
- 最近一次复制的候选人页面文本：`logs/clipboard_dump.txt`

剪贴板转储中可能包含候选人的个人信息。`logs/` 已被 Git 忽略，请勿将日志上传、分享或长期保留，并按照适用的数据保护规则妥善处理候选人信息。

## 常见问题

**找不到 BOSS 直聘窗口**

确认 Edge 已打开目标页面，且窗口标题中包含 `BOSS` 或 `zhipin`。

**关键词始终无法命中**

检查 `logs/clipboard_dump.txt` 是否包含完整的候选人详情文本。如果为空或内容不完整，通常是详情页焦点坐标不正确。

**转发时点击位置偏移**

恢复校准时的分辨率、显示缩放、浏览器缩放和窗口位置，或重新修改坐标常量。

**如何只浏览、不转发**

启动时将关键词留空即可。
